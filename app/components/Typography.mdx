# Typography

The typography system is built around a single idea: hierarchy expressed through type, not color. Two typefaces carry the full weight of the design — Libre Baskerville for headings and Chomsky for the masthead nameplate; Source Serif 4 for everything else. Both are serif, which gives the application its editorial, newspaper-adjacent character.

Three components implement the system: [`Heading`](?path=/docs/components-heading--docs), [`Text`](?path=/docs/components-text--docs), and [`Currency`](?path=/docs/components-currency--docs).
